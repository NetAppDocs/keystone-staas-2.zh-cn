= 根据承诺容量计费
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
承诺容量是购买订阅时为特定性能服务级别承诺的容量。

承诺容量可以是您和NetApp或合作伙伴所接受的单个订阅中各种性能服务级别的总容量。每个Keystone订单上都会注明此容量，并且会计费，无论实际容量消耗是多少。

有关Keystone支持的不同容量的信息，请参阅link:../concepts/supported-storage-capacity.html["Keystone支持的存储容量"]。
